## Puppeteer

https://juejin.cn/post/6975282946638872584

Puppeteer 是一个 Node 库，它提供了一个高级 API 来通过 DevTools 协议控制 Chromium 或 Chrome。

就跟它在 GitHub 简介中介绍的一样：你在浏览器中手动执行的绝大多数操作都可以使用 Puppeteer 来完成！

- 抓取页面快照
- 生成页面 PDF
- 自动操作页面 DOM
