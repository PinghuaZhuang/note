# axios

```js
axios.get(url, {
    headers: {
        'Content-Type': 'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet', // xlsx
    },
    responseType: 'blob',
})
```

responseType的常见类型:

1. blob
2. arraybuffer
3. json
4. text
5. document
6. application/vnd.openxmlformats-officedocument.spreadsheetml.sheet
